# Prompting_payers_project

Este repositorio contiene la documentación de IA (LLM, RAG, NLP, Vector DB, Chunks, Tokens y Prompting) exportada desde mi workspace personal de Notion, la cual contiene la información recopilada durante los meses de 2025 que tuvo lugar la investigación de este tema.

## Estructura

- `docs/`: Contenido principal con imagenes y notas aleatorias.
  - `index.md`: Página de inicio con un índice agrupado por temas (LLM, RAG, Prompting, etc.).
  - Otros `.md`: Páginas individuales de la investigación.
  - `assets/icons/`: Carpeta para íconos personalizados.
- `_config.yml`: Configuración mínima para GitHub Pages usando el tema `jekyll-theme-cayman`.


La página de inicio es `docs/index.md`, que contiene el índice con enlaces al resto del contenido.
